Simple Mech datapack made by Stoddle.
----------------------------------------
Thank you for downloading the datapack! Down below is a list of useful commands. Cheers!

#Spawn a mech at your location
/function core:mech/spawn_mech

#Give yourself every item in the pack
/function core:items/give_items

#Destroy the nearest mech
/function core:mech/remove_nearest_mech

#Run this command to uninstall
/function core:uninstall
